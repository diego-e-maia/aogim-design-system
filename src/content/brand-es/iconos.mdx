## Estilo

- **Trazo:** 2px (stroke-width)
- **Formato:** Outline
- **Esquinas:** Redondeadas
- **Tamaño base:** 24x24px
- **Formato:** SVG inline

## Buenas Prácticas

- Use SVG inline siempre que sea posible para mejor rendimiento y control de color
- Los iconos deben heredar el color del texto mediante `currentColor`
- Mantenga `viewBox="0 0 24 24"` para consistencia
- Prefiera Heroicons o Lucide para nuevos proyectos
