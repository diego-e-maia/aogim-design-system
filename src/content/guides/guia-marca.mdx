## Cores Principais

| Cor | Hexadecimal | Uso |
|-----|------------|-----|
| Laranja AOGIM | `#d68020` | Títulos, botões, elementos de destaque |
| Roxo Espiritual | `#8b55ff` | Detalhes, backgrounds, secundário |
| Fundo Claro | `#f6f6f7` | Backgrounds em modo claro |
| Fundo Escuro | `#0d0d12` | Backgrounds em modo escuro |

## Tipografia

- **Títulos:** Inter Bold, tamanhos 2xl a 6xl
- **Corpo:** Inter Regular, tamanho base (1rem) ou lg (1.125rem)
- **Código:** JetBrains Mono
- **Hierarquia:** h1 → 5xl, h2 → 3xl, h3 → 2xl

## Logotipo

- Use sempre os arquivos SVG disponíveis em Downloads
- Não distorça, rotacione ou aplique efeitos
- Mantenha área de proteção equivalente à altura do "A"
- Prefira fundo escuro para o logo com gradiente

## Espaçamento

Sempre use múltiplos de 4px (0.25rem):

| Token | Pixels | Uso |
|-------|--------|-----|
| 4 | 16px | Padding de cards |
| 6 | 24px | Padding de containers |
| 8 | 32px | Margem entre seções |
| 12 | 48px | Seções grandes |
