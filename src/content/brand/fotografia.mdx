## Diretrizes Técnicas

- **Resolução mínima:** 1920x1080px para uso em tela
- **Formato:** JPEG qualidade 90% ou PNG para imagens com transparência
- **Proporção:** 16:9 para widescreen, 4:3 para impressos
- **Cor:** Perfil sRGB
